# OneQueryToRuleThemAll
KQL queries I’ve written, found, tweaked, broken, fixed, and reused more times than I care to admit. Mainly used for hunting, investigations, and figuring out why the logs are mad today.

*Not all of these Queries are my own creation, if I have gotten them from another person in the community, I will give them credit within the actual description of the query.* 
<br>

This project is also ongoing, I am continuously porting over queries I use daily into this Repo, chances are if you do not see a query useful to you, come back in a few days or so and it may be added.
